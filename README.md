# Виселица

### Команды для запуска

1. Запустить приложение:
    ```shell
    python hangman.py
    ```
2. Запустить авто-форматирование кода (`black`):
    ```shell
    black .
    ```
3. Запустить проверку кода (`ruff`):
    ```shell
    ruff check .
    ```
4. Запустить проверку типов (`mypy`):
    ```shell
    black .
    ```
